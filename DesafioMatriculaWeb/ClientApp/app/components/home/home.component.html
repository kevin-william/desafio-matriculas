Verificador de matrículas

Essa aplicação foi desenvolvida como solução para o desafio da "empresa XPTO".

Esta solução se propõe a verificar matrículas de determinado arquivo e possui duas funções:

Gerar Digitos Verificadores:

Para gerar os dígitos verificadores,é necessário que o usuário acesse a guia "Gerar Dígitos Verificadores" e envie seu arquivo. O mesmo precisa de matrículas compostas por 4 dígitos numéricos. Cada linha do arquivo contém uma matrícula. Após fazer o upload, o arquivo será processado e um novo arquivo contendo as matrículas com seus respectivos dígitos vereficadores será criado.

Verificar Matrículas:

O sistema recebe um arquivo contendo matriculas. Cada linha do arquivo possui uma matricula composta por 4 dígitos numéricos, um "-"(traço) e um dígito Hexadecimal. Para realizar a verificação, é necessário que o usuário acesse a guia "Verificar Matrículas" e envie seu arquivo. Nesse momento o sistema analisará se cada matrícula possui o dígito verificador válido. Será criado um novo arquivo contendo as matriculas verificadas seguido por "verdadeiro" para as matrículas que contém o dígito verificador válido e "falso" para aquelas que possuem o dígito verificador inválido.
